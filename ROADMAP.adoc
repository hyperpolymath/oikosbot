This repository contains *two separate products*. They each have their own
roadmap.

[cols="1,3", options="header"]
|===
| Product | Roadmap

| *Oikos* (the DSL)
| link:docs/ROADMAP.adoc[`docs/ROADMAP.adoc`] — typed compiler for stock-flow
  consistent macroeconomic models.
  Currently `v0.1.0-dev` scaffold; v0.3 milestone is blocked on the Ephapax
  parser milestone.

| *OikosBot* (the GitHub App)
| link:bot-integration/ROADMAP.md[`bot-integration/ROADMAP.md`] — code-analysis
  bot at https://github.com/apps/oikosbot[`apps/oikosbot`].
  v0.1 (hardened scaffold) just landed; v0.2 (first production install) is
  blocked on a domain + deployment decision.
|===

See link:docs/disambiguation/oikos-dsl-vs-oikosbot.adoc[`docs/disambiguation/oikos-dsl-vs-oikosbot.adoc`]
for why these are different products despite the shared name.
