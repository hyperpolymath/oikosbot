= OikosBot Quickstart
:icons: font
:source-highlighter: rouge

== Build

[source,bash]
----
cargo build --release
----

== Usage

=== Analyze a file

[source,bash]
----
./target/release/oikosbot analyze path/to/file.rs
----

Supports: Rust (`.rs`), JavaScript (`.js`, `.mjs`), TypeScript (`.ts`).

=== Analyze oikosbot itself (dogfooding!)

[source,bash]
----
./target/release/oikosbot self-analyze
----

This proves we practice what we preach — the analyzer itself is efficient.

=== JSON output

[source,bash]
----
./target/release/oikosbot analyze file.rs --format json
----

== What It Does

OikosBot analyzes code for:

* *Energy consumption* (Joules)
* *Carbon footprint* (gCO2e)
* *Execution time* (milliseconds)
* *Memory usage* (bytes)

And provides:

* *Eco score* (0-100) — environmental impact
* *Econ score* (0-100) — algorithmic efficiency
* *Quality score* (0-100) — code quality
* *Overall health* (weighted combination)
