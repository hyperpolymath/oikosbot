== Status

`0.1.0-dev` - scaffold only.

The AffineScript webhook receiver is in flight. Cross-module type-checking,
JSON payload extraction, and the HTTP server accept loop are still gated on
upstream AffineScript standard-library work. The Haskell analyzer scaffold is
present and builds independently.

OikosBot is not currently deployed in production.

== Repository Layout

[source,text]
----
oikosbot/
|-- bot-integration-affine/  # AffineScript webhook receiver
|-- analyzers/code-haskell/  # Haskell eco/econ/quality analysis engine
|-- policy-engine/           # Datalog and DeepProbLog policy rules
|-- config/                  # Bot configuration
|-- prompts/                 # Generated assistant prompt templates
|-- database/                # Knowledge-graph schema and query assets
|-- databases/               # Legacy database assets retained for migration
|-- docs/GITHUB_APP_SETUP.md # GitHub App registration notes
|-- ARCHITECTURE.md          # Platform architecture
`-- DEPLOY.md                # Deployment status and runbook notes
----

== Local Checks

[source,shell]
----
just haskell-build
just haskell-test
just affine-check
----

`just affine-check` expects an AffineScript compiler. Override the compiler path
with `AS_BIN=/path/to/main.exe` when needed.
